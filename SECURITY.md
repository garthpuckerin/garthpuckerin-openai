# Security Policy

## Supported Versions

Currently only the `main` branch is actively supported with zero-day vulnerability checks and fixes.

| Version | Supported          |
| ------- | ------------------ |
| v1.x    | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

We treat the security and safety of our infrastructure critically. CodeQL SAST scanning and Dependabot are actively monitoring for vulnerabilities.

If you suspect you have found a zero-day issue or bypass that bypasses the automation, please immediately report it to **garth.puckerin@gmail.com**.

Do **not** create a public issue regarding security vulnerabilities until an advisory has been created, and the vulnerability has been patched. You will receive an initial response to your vulnerability report within 24-48 hours.
